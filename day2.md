# Python基础学习笔记

## 1. Python简介
- Python 是一种高级、解释型、通用编程语言。
- 特点：语法简洁、易读易学、跨平台、可扩展。
- 应用领域：Web开发、数据分析、人工智能、自动化脚本等。

## 2. 环境搭建
- **安装Python**：从 [python.org](https://www.python.org) 下载最新版本，推荐 3.x 版本。
- **IDE选择**：
  - PyCharm：功能强大，适合大型项目。
  - VS Code：轻量，支持Python插件。
  - IDLE：Python自带，适合初学者。
- **验证安装**：
  ```bash
  python --version
  ```
